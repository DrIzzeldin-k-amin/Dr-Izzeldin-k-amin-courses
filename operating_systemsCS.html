General Objectives of Course

Brief description of Course: This course deepens the understanding of virtualization, concurrency, and persistence. It explores the key functions of an operating system and enables students to develop applications using system calls.

General Objectives:

  • Understand different internal structures of operating systems for various hardware.
  • Learn about concurrency, processes, and thread management, along with inter-process communication.
  • Study process synchronization and deadlocks.
  • Understand CPU scheduling and memory management techniques, including virtual memory.
  • Explore mass storage structures, I/O systems, and file system interfaces.
  • Examine operating system examples, including MS Windows, Linux, and Android.
  • Learn about protection and security in operating systems.
Intended Learning Outcomes (ILOs)
  • Knowledge and Understanding: Understand the expected functions of an operating system and how to use them.
  • Intellectual Skill: Learn the mechanisms of using key operating system functions and new developments in the field.
  • Professional and Practical Skills: Develop system programming skills, creating applications that utilize key operating system functions.
  • General and Transferable Skills: Understand how to install and configure an operating system, and master adding new features or drivers.
